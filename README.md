# cyber-defense-game

Cyber Defense is a small neon tower defense game where turrets automatically target and destroy incoming enemies while you manage upgrades.

This was built and shipped as a short challenge (about 3 hours total), focusing on getting a full gameplay loop working and deployed.

Game can be found at: https://devwithchaos.itch.io/cyber-defense

The current version is v1 - a functioning loop where enemies spawn, turrets engage automatically, and the player upgrades fire rate, spawn rate, and detection radius over time.

It plays more like an idle/observer experience right now, with the player influencing the system through upgrades rather than direct control.
